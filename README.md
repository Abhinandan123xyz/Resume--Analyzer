# AI Resume Analyzer

An AI-powered resume analyzer built with Python, Streamlit, and Groq API.

## Features
- Resume scoring out of 100
- Job description match percentage
- Missing skills and keyword detection
- Strengths and weaknesses analysis
- PDF upload and text input support

## Tech Stack
Python, Streamlit, Groq API (Llama 3.3 70B), PyMuPDF

## Run Locally
pip install -r requirements.txt
streamlit run app.py
